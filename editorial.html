このサイトの約束

編集方針

財政や国債は、立場によって説明の仕方が大きく変わるテーマです。このサイトでは、結論を押しつけるのではなく、読者が自分で考えるための材料を整理します。

一つの数字ではなく、
一つの見方でもなく。
1

一つの数字だけで結論を出さない

政府債務だけでなく、金利、税収、経済成長、資産、保有者、通貨制度なども確認します。

2

一つの立場だけで説明しない

国債の強みと制約、政府の負債と保有者の資産など、複数の面を並べます。

3

事実・解釈・意見を分ける

公的統計で確認できる事実と、評価が分かれる解釈を混同しません。

4

分からないことは断定しない

将来予測や評価が分かれる問題は、前提と不確実性を明記します。

出典と数字の扱い

✓ 財務省・日本銀行など一次資料を優先
✓ 数字には基準日または基準年を表示
✓ 実績・予算・見込み・予測を区別
✓ 定義が異なる数字を直接比べない
✓ 資料名と公表機関を明記
✓ 重要な修正は更新履歴に残す

表現の方針

避ける表現

「日本は必ず破綻する」
「絶対に破綻しない」
「国債は無限に発行できる」
「政府資産があるので問題ない」

このサイトの書き方

「自国通貨建てには強みがある」
「インフレや金利も確認する」
「政府資産には性質の違いがある」
「複数の指標を合わせて見る」

初心者への配慮

✓ 専門用語には短い説明を付ける
✓ 一文を長くしすぎない
✓ 図だけでも基本構造が分かるようにする
✓ 読者を怖がらせない
✓ 読者を馬鹿にしない
✓ 家計と国家財政を完全に同じものとして扱わない
