# 財政の見方

国債・税金・政府の資産を、初心者向けに図と短い文章で説明する静的サイトです。

## 公開方法

### GitHub Pages
1. このフォルダの中身をGitHubリポジトリのルートへ配置
2. Settings → Pages
3. Deploy from a branch を選択
4. main / root を指定

### Netlify
このフォルダをそのままドラッグ＆ドロップして公開できます。

## 主なページ

- `index.html`：トップページ
- `article-01-kokusai.html` ～ `article-09-comparison.html`：基本記事
- `glossary.html`：用語集
- `sources.html`：主な出典・資料
- `editorial.html`：編集方針
- `roadmap.html`：今後追加する内容
- `site-map.html`：サイトマップ

## 公開前に行うこと

公開URLが決まった後で、次を追加します。

- `sitemap.xml` の絶対URL
- `robots.txt` のSitemap行
- 各ページのcanonical URL
- OGP画像とSNS共有設定
